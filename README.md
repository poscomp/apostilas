apostilas
=========
Para os nomes dos arquivos desta pasta, favor manter o padrão:

Arquivos pdf: [Código do tópico¹]_[nomeDoGrupo²].pdf
Arquivos tex: [Código do tópico¹]_[nomeDoGrupo²].tex
Arquivos bib: [Código do tópico¹]_[nomeDoGrupo²].bib
Analogamente para outros tipos de arquivo...

¹Acesse a lista de tópicos.
²Nome do grupo: matematica, fundamentosDaComputacao ou tecnologiaDaComputacao.
